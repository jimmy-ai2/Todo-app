# TaskFlow: Todo App

A lightweight, responsive **Task Management application** built with modern web technologies.  
TaskFlow helps users efficiently manage daily tasks with **real-time filtering** and **persistent storage**.

---

## Features

- **Task Management**  
  Add, toggle completion, and delete tasks بسهولة.

- **State Persistence**  
  Tasks are saved using **localStorage**, so they remain after refreshing.

- **Dynamic Filtering**  
  View tasks by:
  - All
  - Active
  - Completed

- **Progress Tracking**  
  Real-time counter showing remaining active tasks.

- **Bulk Cleanup**  
  Remove all completed tasks with one click.

- **Contextual Date**  
  Displays the current day and date automatically.

- **Visual Feedback**  
  Shows an empty state message when no tasks exist.

## Tech Stack

- **HTML5** – Semantic structure
- **CSS3** – Styling with Flexbox, CSS Variables & responsive design
- **JavaScript (ES6+)** – Core logic, DOM manipulation, state management

## How to Use

1. **Add a Task**  
   Type in the input field and press `Enter` or click the ➕ button.

2. **Complete a Task**  
   Click the checkbox next to a task.

3. **Filter Tasks**  
   Switch between:
   - All
   - Active
   - Completed

4. **Delete a Task**  
   Hover and click the ❌ button.

5. **Clear Completed**  
   Remove all finished tasks at once.
